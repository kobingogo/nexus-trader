## 1. Product Vision (产品愿景)

NEXUS Trader (A-Share Edition) 旨在成为 A 股投资者的**"理性副驾驶" (Rational Co-pilot)**。
V1.0 版本核心目标：**通过数据可视化、实时异动监控与 AI Agent 协作，帮助用户快速识别市场情绪周期与资金流向，规避非理性交易。**

**核心价值主张：**
*   **降噪 (Noise Cancellation):** 过滤无效波动，只提示关键异动。
*   **共振 (Resonance):** 寻找情绪与资金的共振点（买点）。
*   **风控 (Risk Control):** 识别情绪退潮期（卖点）。
*   **智能 (Agentic):** 从"人找数据"转变为"数据找人"，通过自然语言交互实现深度洞察。

---

## 2. Target User (目标用户)

*   **短线/超短线选手：** 关注打板、接力、热点题材切换。
*   **趋势交易者：** 关注资金流向、板块轮动。
*   **痛点：** 信息过载，盘中无法快速判断情绪冰点/沸点，容易追涨杀跌。

---

## 3. Scope (功能范围) - V1.0 MVP

本版本专注于**"看" (Monitoring)** 与 **"警示" (Alerting)**，暂不涉及自动交易。

### 3.1 核心功能模块

#### A. 市场情绪仪表盘 (Market Mood Dashboard)
*   **全市场温度计 (0-100):**
    *   **极寒 (0-20):** 冰点期，大面积跌停，亏钱效应显著。（策略：空仓/试错）
    *   **回暖 (20-50):** 修复期，跌停减少，首板增多。（策略：低吸核心）
    *   **沸点 (80-100):** 高潮期，百股涨停，连板高度突破。（策略：加速/兑现）
*   **关键指标：**
    *   涨跌家数比 (Up/Down Ratio)
    *   涨停/跌停家数 (Limit Up/Down Count)
    *   昨涨停今表现 (Premium Rate): 昨日打板的人今天赚钱了吗？
    *   炸板率 (Broken Limit Rate): 封板意愿强不强？

#### B. 异动精灵 (Anomaly Detector) - "NEXUS Eyes"
*   **主力抢筹监控：** 盘中实时扫描：
    *   大单买入 > 1000万
    *   快速拉升 > 3% / 1分钟
    *   **NEXUS 评语：** "主力点火，关注承接。"
*   **主力出逃预警：**
    *   大单卖出 > 1000万
    *   高位跳水 > 3% / 1分钟
    *   **NEXUS 评语：** "高位核按钮，撤退！"
*   **集合竞价异动 (9:15-9:25):** 谁在竞价阶段抢跑？谁在最后时刻撤单？

#### C. 题材热度风向标 (Sector Heat Map)
*   **实时资金流向：** 哪个板块吸金最多？（如：低空经济、AI、中特估）
*   **板块轮动图：** 展示资金从 A 板块流向 B 板块的路径。
*   **龙头股联动：** 龙头涨停，龙二、龙三是谁？

#### D. 复盘助手 (Daily Review)
*   **每日复盘报告：** 自动生成今日市场总结（Markdown 格式）。
    *   今日最强板块
    *   今日亏钱效应源头
    *   NEXUS 建议：明日策略（防守/进攻）。

### 3.2 AI Agent Capabilities (Phase 1: Copilot)

#### E. 智能交互中枢 (NEXUS Copilot)
*   **自然语言查询 (NL2SQL/API):**
    *   用户提问："今天最强的板块是哪个？" -> Agent 调用 `get_top_sector()` -> 回答："低空经济，涨幅 5.2%。"
    *   用户提问："帮我盯着万丰奥威，如果涨停告诉我。" -> Agent 设置 `Condition Alert`。
*   **主动式情报推送 (Proactive Intel):**
    *   **新闻关联：** 监控到新闻（如"低空经济"）-> 自动关联板块/个股 -> 推送："利好突发！低空经济板块异动，建议关注龙头 XXX。"
    *   **情绪预警：** 当大盘情绪急剧下降（炸板率飙升）-> 主动弹窗："警告：市场情绪退潮，建议停止开仓！"

## 4. User Journey (用户旅程)

1.  **盘前 (9:00):** 打开 NEXUS，查看外围市场影响 + 竞价异动预判。
2.  **盘中 (9:30-15:00):**
    *   **挂机模式：** NEXUS 监控全市场，无异动不打扰。
    *   **异动触发：** "滴！低空经济板块资金净流入加速，龙头万丰奥威即将封板。"
    *   **决策：** 用户查看 K 线与筹码结构，决定是否跟随。
3.  **盘后 (15:30):** 查看复盘报告，验证今日操作逻辑。

---

## 5. Non-Functional Requirements (非功能需求)

*   **实时性 (Latency):** 数据延迟 < 3秒 (依赖数据源)。
*   **稳定性 (Reliability):** 盘中 4 小时零宕机。
*   **UI/UX:** 深色极简风格 (Dark Mode)，红绿配色符合 A 股习惯（红涨绿跌），信息密度高但不杂乱。

---

## 6. Success Metrics (成功指标)

*   **准确率：** 异动提示的有效性（提示后上涨概率 > 60%）。
*   **用户留存：** 用户是否每天盘中都开着它。
